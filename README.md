# 🍽️ Culinary Curator: AI Restaurant Recommender

**Culinary Curator** is a modern, AI-powered restaurant recommendation engine that combines deterministic business logic with the power of Large Language Models (LLMs) to provide a premium, personalized dining discovery experience.

---

## ✨ Key Features

-   **🎯 Hybrid Recommendation Engine**: Uses a deterministic Filtering Engine (Pandas) for accuracy and an LLM (Groq) for personalized ranking and reasoning.
-   **🤖 AI Insights**: Every restaurant card includes a custom, context-aware explanation generated by AI.
-   **🔍 Searchable Discovery**: Type-to-search dropdowns for 90+ Bangalore neighborhoods and 100+ cuisines.
-   **⚡ Live Filtering**: Instant client-side sorting by *Top Rated*, *Price*, and *AI Recommendation*.

---

## 🚀 Getting Started

### **1. Prerequisites**
-   Python 3.10+
-   Node.js 18+
-   A **Groq API Key**

### **2. Backend Setup (FastAPI)**
Navigate to the **`backend/`** directory:
```bash
cd backend

# Install dependencies
pip install -r requirements.txt

# Start the API
python -m uvicorn restaurant_rec.phase4.app:app --reload --app-dir src
```

### **3. Frontend Setup (Next.js)**
Navigate to the **`frontend/`** directory:
```bash
cd frontend

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Visit **[http://localhost:3000](http://localhost:3000)** to start exploring!

---

## 📂 Project Structure

-   `backend/src/`: Core logic (Filters, LLM reasoning, API endpoints).
-   `backend/data/`: Optimized Parquet datasets.
-   `frontend/`: Next.js application.
